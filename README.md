# Lumière — Салон красоты

Современный одностраничный сайт для премиального салона красоты в тёмных тонах.

## Стек

- React 18
- TypeScript
- Styled Components
- Framer Motion (анимации)
- Vite

## Запуск

```bash
npm install
npm run dev
```

Сайт откроется на [http://localhost:5173](http://localhost:5173)

## Сборка

```bash
npm run build
npm run preview
```

## Структура

- **Header** — фиксированное меню с мобильной навигацией
- **Hero** — главный экран с parallax-эффектом
- **Services** — карточки услуг с hover-анимацией
- **About** — блок «О нас» с ценностями салона
- **Contact** — форма обратной связи с валидацией
- **Footer** — контакты и соцсети
